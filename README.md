# Лабораторная работа №3 C#

Разработана система для управления заказами в службе доставки еды.

Система выполняет:

1. Поддерживает создание и управление заказами, включая возможность выбора блюд из меню.
2. Обрабатывает различные типы заказов, такие как стандартные заказы и заказы с особенными условиями (например, быстрая доставка, персональные предпочтения).
3. Предоставляет возможность отслеживания состояния заказа: подготовка, доставка, выполнен.
4. Включает функции для расчета стоимости заказа с учетом различных факторов, таких как скидки, налоги, плата за доставку.
5. Логика покрыта unit тестами
6. CLI не нужен
7. Использовано не менее 5 паттернов ООП, изученных за курс. Иметь четкое обоснование почему был выбран каждый 

Система разработана для стартапа в сфере доставки еды, который хочет обеспечить быструю и надежную обработку заказов. Система масштабируема, чтобы справляться с растущим числом пользователей.

Цель ЛР заключается в изучении и применении паттернов проектирования для создания гибкой и расширяемой архитектуры, особенно в условиях, когда требования могут быстро изменяться.
